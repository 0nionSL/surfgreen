# SurfGreen

Mobile app for surfers — find the best waves at a glance.

## Structure

```
surfgreen/
├── backend/     # FastAPI API (Python 3.11+)
└── frontend/    # React Native + Expo (coming in Stage 2)
```

## Getting started

See [backend/README.md](backend/README.md) for API setup and deployment.

## MVP regions

- **Bali:** Kuta, Canggu, Keramas, Uluwatu, Bingin
- **Portugal:** Ericeira, Supertubos, Armacao de Pera, Sagres, Carvoeiro

## Score legend

| Color | Score | Meaning |
|-------|-------|---------|
| Green | 80–100 | Ideal conditions |
| Yellow | 60–79 | Rideable |
| Red | <60 | Poor conditions |
